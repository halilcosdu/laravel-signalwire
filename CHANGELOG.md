# Changelog

All notable changes to `laravel-signalwire` will be documented in this file.

## v1.1.0 - 2026-04-09

### What's Changed

* Added Laravel 13 support
* Updated dev dependencies for broader version compatibility (Pest 3, PHPStan 2, Larastan 3, Orchestra Testbench 10/11)
* Updated CI workflow to test against Laravel 13.x with Orchestra Testbench 11.x
* Package now supports Laravel 10.x, 11.x, 12.x, and 13.x
